# Alerts Documentation

This folder contains documentation and resources related to alerts in the Mobikwik system.

## Purpose
The alerts folder documents what to check after alerts are triggered and the actions required to identify the culprit.

## Screenshots
Screenshots of alerts are attached here. The naming convention for screenshots and Draw.io images is kept consistent for easy identification.

## Flow Diagrams
The `flow` folder contains diagrams illustrating the steps to check after alerts are received, using various Draw.io images.
